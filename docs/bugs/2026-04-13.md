# 2026-04-13 问题登记

## bug 处理流程
**分析登记信息** - 逐个问题进行分析
**登记issue** - 使用gh cli登记issue：记录问题产生的原因、修复方案、建议测试方案
**添加issue-list** - issue-list.md文档中添加此issue的标题及状态
**修复issue** - 修复问题，代码提交后,评论区备注commit信息，将issue标记为待测试验证状态
**打勾** - 修改此文档的相应项，打勾；
**修改issue-list** - 修改issue-list.md中此issue的处理状态
